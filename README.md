# wedding-invitation
Проект представляет собой свадебное онлайн-приглашение на торжество по случаю свадьбы родного старшего брата Даанышмана и его невесты Нуржана. Сайт оформлен в праздничном стиле и содержит основную информацию: дату, время и приглашение для гостей. Цель проекта — красиво и удобно передать детали и создать атмосферу праздника ✨
